Vulnerabilidades Identificadas

Alta Clave de AWS expuesta en código
src/config.js:15

Descripción del hallazgo

Se ha detectado una clave de acceso secreta de AWS expuesta directamente en texto plano dentro del código de producción. Esto permite a cualquier atacante con acceso al repositorio comprometer los recursos de infraestructura de AWS.

Remediación recomendada

Mueve inmediatamente la credencial a un archivo .env y agrégalo a tu .gitignore para evitar subirlo al control de versiones. Asegúrate de rotar las credenciales antiguas en la consola de AWS.

const AWS_KEY = 'AKIAIOSFODNN7EXAMPLE';
